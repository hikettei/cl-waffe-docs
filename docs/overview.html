Overview

Welcome to cl-waffe!

This package is in development.

cl-waffe is a deep learning framework, which is aimed to:

  1. Simple and fast.
  2. Integrate environments.
  3. Extensible for users.

LLA Setting

If BLAS is already set up, skip this.

cl-waffe depends on LLA and you have to set up configurations.

Write the following code (path should be changed depending on the environment.) to your startup script: (e.g.: ~/.sbclrc, ~/.roswell/init.lisp and so on.)

(defvar *lla-configuration*
  '(:libraries ("/usr/local/opt/openblas/lib/libblas.dylib")))

When Memory Exhausted

It is better to run this command in advance, since training a model requires a lot of memory.

$ ros config set dynamic-space-size 4gb

In SLIME and SBCL:

Please add:

(setq slime-lisp-implementations '(("sbcl" ("sbcl" "--dynamic-space-size" "4096"))))
